# 开发运行与验证说明

更新时间：2026-06-15

## 1. 当前实现范围

当前工程已经搭建为 TypeScript workspace：

```text
apps/mobile    Android Expo 应用骨架
apps/server    本地 Fastify + SQLite 服务端
apps/web       电脑 Web 管理与学习界面
packages/shared 共享类型、CSV 工具、SRS 复习算法
```

已实现的基本可用能力：

- Web 端连接本地服务并显示复习台、词库、CSV 导入导出、资源包和同步诊断。
- 服务端使用 SQLite 文件库存储单词、词义、音频元数据、卡片、复习记录、同步设备和资源包。
- 支持新增、删除、搜索单词。
- 支持 CSV 导入导出。
- 支持到期复习队列和 `again / hard / good / easy` 复习结果。
- 支持统计：总词数、待复习、今日复习、保持率、缺失音频。
- 支持局域网协议入口：`/health`、`/.well-known/elsync`、`/pair`、`/sync/changes`、`/sync/apply`。
- Android Expo 应用具备离线 SQLite 建表、新增单词、复习、连接电脑端健康检查的第一版骨架。

## 2. 安装依赖

```bash
npm install
```

## 3. 常用命令

```bash
npm run typecheck
npm test
npm run build
```

导入基础词汇与本地发音资源包：

```bash
npm run import:basic-vocabulary -- --limit 300
```

补指定词的发音资料：

```bash
npm run import:basic-vocabulary -- --words persist,verify
```

启动构建后的本地服务：

```bash
npm --workspace @els/server start
```

如果需要让服务在 Codex 临时命令结束后仍然保持可访问，使用 macOS 用户级服务托管：

```bash
launchctl remove com.yanghuanyu.english-learning-system.server >/dev/null 2>&1 || true
launchctl submit -l com.yanghuanyu.english-learning-system.server \
  -o /tmp/英语学习系统-server.log \
  -e /tmp/英语学习系统-server.err.log \
  -- /bin/zsh -lc 'cd "/Users/yanghuanyu/codex/English Learning System" && exec node --no-warnings apps/server/dist/index.js'
```

停止托管服务：

```bash
launchctl remove com.yanghuanyu.english-learning-system.server
```

检查服务状态：

```bash
launchctl list | grep com.yanghuanyu.english-learning-system.server
curl http://127.0.0.1:5174/health
```

服务端默认监听：

```text
http://127.0.0.1:5174
http://本机局域网 IP:5174
```

启动 Web 开发服务：

```bash
npm --workspace @els/web run dev
```

启动 Android Expo 开发服务：

```bash
npm --workspace @els/mobile run dev
```

连接 Android 真机或模拟器后运行：

```bash
npm --workspace @els/mobile run android
```

## 5. 已执行验证

已通过：

- `npm --workspace @els/shared test`
- `npm --workspace @els/server test`
- `npm test`
- `npm run typecheck`
- `npm run build`
- `npm --workspace @els/mobile exec -- expo config --type public --json`
- `npm --workspace @els/mobile exec -- expo export --platform android --output-dir dist-android`

已执行 HTTP 冒烟：

- `GET /health` 返回服务状态。
- `POST /words` 可新增单词。
- `GET /reviews/due` 可返回待复习卡片。
- `POST /reviews/:cardId` 可记录复习结果并更新 SRS 状态。
- `GET /stats` 可返回统计。
- `POST /imports/csv` 可导入 CSV 并报告缺失音频。
- `GET /exports/csv` 可导出词库。
- `GET /.well-known/elsync` 可返回局域网同步协议信息。
- `POST /pair` 可生成配对 token。
- `GET /sync/changes?since=0` 可返回同步变更。

已执行 Web 验证：

- 构建后的服务端首页 `GET /` 返回 Web HTML。
- Web 构建资源 `/web-assets/*.js` 和 `/web-assets/*.css` 返回 200。
- 内置浏览器实际渲染出页面标题、侧边栏、统计数据、复习卡片和复习按钮。
- 已导入基础词汇资源包 `basic-dale-chall-foundation-2026-06-16`，新增 300 个基础词和 300 个本地发音文件。
- 已用 `pronunciation-backfill-2026-06-16` 补齐旧词发音；当前 `audio_assets.missing = 0`。

Android 当前验证边界：

- 已通过 TypeScript 类型检查。
- 已通过 Expo 配置解析，确认应用名、slug、Android 包名和局域网权限。
- 已通过 Expo Android 平台 JS bundle 导出。
- 尚未在本机完成 Android 真机或模拟器安装运行；需要后续连接 Android 环境后执行 `npm --workspace @els/mobile run android`。
