## 2. 安装方式

如果手机已开启 USB 调试，并且电脑能识别设备：

```bash
./.android-sdk/platform-tools/adb devices
./.android-sdk/platform-tools/adb install -r "outputs/英语学习系统-安卓调试版.apk"
```

也可以把 APK 文件传到 Android 手机上，点击安装。首次安装调试 APK 时，Android 可能要求允许“安装未知来源应用”。

## 4. 重新打包

```bash
npm run build:android:debug
```

重新打包后的 APK 默认位置：

```text
apps/mobile/android/app/build/outputs/apk/debug/app-debug.apk
```

如果需要交付给手机安装，可复制到 `outputs/`：

```bash
cp "apps/mobile/android/app/build/outputs/apk/debug/app-debug.apk" "outputs/英语学习系统-安卓调试版.apk"
```

## 6. 注意事项

- 当前 APK 是 debug 版本，适合本机安装测试，不适合正式发布。
- 这是本地离线优先 App 骨架，电脑端同步功能仍需后续在 Android 真机上继续联调。
- 如果手机提示覆盖安装失败，先卸载旧版 `com.local.englishlearningsystem` 后再安装。
